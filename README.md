## 📝 Descrição
Este projeto é um ambiente de Realidade Virtual básico, criado para demonstrar os fundamentos de XR utilizando a engine Unity. O foco principal é a aplicação prática de configurações de ambiente, posicionamento espacial de objetos 3D e navegação pelo cenário.

---

## 🕹️ Como testar a movimentação
Siga os passos abaixo para testar o projeto diretamente no seu computador (Unity Editor), sem a necessidade de um headset de Realidade Virtual:

1. Abra a cena principal localizada na pasta `Assets/Scenes`.
2. Pressione o botão **Play** na parte superior central do Unity Editor.
3. Utilize as teclas **W, A, S, D** (ou as setas direcionais) para caminhar pelo cenário.
4. Movimente o **Mouse** para girar a câmera e explorar o ambiente ao redor.
